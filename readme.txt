== Description ==

This plugin are display different-different navigation on each page and post.You Can easily setup different navigation or menu on pages and single posts. please follow these steps:

Step 1: First click on "WP Different Navigation Box Settings" sub menu in settings menu in wordpress admin section. Then select Posttypes. You want to display "Different Navigation Box" MetaBox on Posttypes.

Step 2: Select Menu in "Different Navigation Box" Metabox for Current Page/Post Display Menu.

Step 3: "WP Different Navigation Widget" put this widget you want display menu in widget area. and enter widget title and select default menu and click on save button.



== Installation ==
Automatic installation through WordPress:

   1. Log-in to your WordPress site.
   2. Hover over Plugins and click Add New.
   3. Under Search type in wp-different-navigation-on-each-page-and-post, then click Search Plugins.
   4. In the results page, click Install Now.
   5. Once installed, click Activate Plugin. You're done!

Manual installation:

   1. Upload the 'wp-different-navigation-on-each-page-and-post' directory to the /wp-content/plugins/ directory.
   2. Go to your WordPress dashboard and navigate to Plugins -> Installed Plugins.
   3. On the Plugin page in your WordPress Administration area, activate the wp-different-navigation-on-each-page-and-post plugin.
